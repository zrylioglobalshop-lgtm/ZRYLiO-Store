# ZRYLiO Store - E-Commerce Platform

**একটি সম্পূর্ণ ফাংশনাল ই-কমার্স প্ল্যাটফর্ম**

## ফিচার:
- ✅ মাল্টি-ক্যাটেগরি পণ্য (ইলেকট্রনিক্স, ফ্যাশন, সবকিছু)
- ✅ কাস্টমার সাইনআপ/লগইন
- ✅ শপিং কার্ট সিস্টেম
- ✅ অর্ডার ম্যানেজমেন্ট
- ✅ এডমিন প্যানেল
- ✅ দেশব্যাপী ডেলিভারি (বাংলাদেশ, ইটালি)
- ✅ রেসপন্সিভ ডিজাইন

## ইনস্টলেশন:
1. ডাটাবেস ইমপোর্ট করুন: `database.sql`
2. `config.php` এ ডাটাবেস কানেকশন সেট করুন
3. ওয়েব ব্রাউজারে খুলুন

## ফোল্ডার স্ট্রাকচার:
```
ZRYLIO-Store/
├── index.php (হোমপেজ)
├── products.php (পণ্য লিস্ট)
├── product-details.php (পণ্য বিস্তারিত)
├── cart.php (কার্ট)
├── checkout.php (চেকআউট)
├── customer-dashboard.php (কাস্টমার ড্যাশবোর্ড)
├── admin/ (এডমিন প্যানেল)
├── css/ (স্টাইল ফাইল)
├── js/ (জাভাস্ক্রিপ্ট)
└── config.php (কনফিগারেশন)
```
